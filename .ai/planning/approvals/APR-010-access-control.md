# APR-010 — 접근제어(팀별 서가 열람 권한) 승인

- 상태: Pending Approval
- 요청일: 2026-07-08
- 요청자: (오너 지시로 설계됨)
- 관련 ADR: [ADR-010](../../decisions/ADR-010-team-shelf-access-control.md)

## 요청 요약
지식 도서관에 **팀별 서가(component) 열람 권한**을 도입한다. 기본 deny, 단일 시행 계층, 감사 포함.
보안·데이터 접근 정책이라 사람 승인이 필요하다.

## 왜 사람이 결정해야 하나
- 내부 지식 노출 범위를 정하는 결정 — 잘못되면 정보 유출/과잉차단. 되돌리기 비싸다.
- MCP 로컬 신뢰 모델의 한계(민감 서가 노출 여부)를 사람이 판단해야 한다.

## 결정 대상
1. 1단계(정책 필터+감사, 팀은 설정/env 주입)부터 착수할지.
2. 민감 서가를 MCP 에 아예 노출하지 않을지(전용 인스턴스 분리).
3. 팀→서가 매핑의 초기 소유자/관리 주체.

## 영향
- 승인 시: ADR-010 1단계 구현(AccessPolicy + 질의 서가 필터 + 감사 영속화).
- 미승인 시: 현행 유지(전원이 전체 열람) — 착수 불가.
